# 🧪 3 — Evaluation Runs: Live Case Studies

This directory contains the "Lab Reports" of my evaluation work. These are actual side-by-side (SxS) comparisons and reasoning traces from my work on professional annotation platforms.

## 📋 Run Structure

Each run follows a standardized format:
- **Prompt:** The original user instruction.
- **Model A vs Model B:** Parallel outputs.
- **The Reasoning Trace:** My internal monologue as I evaluate.
- **The Verdict:** The final ranking and detailed justification.
