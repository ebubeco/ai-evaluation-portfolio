Edge-Case Taxonomy

Failure modes
I test for

Instruction Hierarchy
Operator-Layer Constraint Override
System context rules violated by user-turn instructions. Often invisible to single-pass raters — the constraint is buried, the response is fluent.
Hallucination
Compatible ≠ Entailed Fabrication
Claims contextually plausible but not grounded in the prompt. Confident framing amplifies the failure. One confirmed hallucination overrides everything.
Reasoning
Correct Answer, Broken Path
Right conclusion reached through invalid inference. A correct answer from broken reasoning is a reasoning failure. The path is evaluated, not just the destination.
Ambiguity
Silent Assumption Injection
Model resolves genuine ambiguity without flagging it. Silent assumption = failure. Explicit assumption + minimal statement = acceptable minimum.
Multimodal
Visual Grounding Failure
Fluent text description of an image that misrepresents spatial relationships, entities, or attributes. Textual fluency is an unreliable quality proxy.
Cultural / Geographic
Training Data Distribution Bias
Systematic quality gap between Western and non-Western knowledge domains. Invisible to US-centric rubrics. Requires parallel prompt batteries to detect.

CASE-001 · 2025
RLHF · Adversarial
Constraint & Instruction Hierarchy Failures
Adversarial TestingConstraint HierarchyRLHF RankingFailure Taxonomy
Output6 failure archetypes documented
Problem
Operator-layer constraints overridden by user-layer preferences. Surface fluency masked the failure. Standard raters were passing outputs that violated explicit exclusions buried in system context.
Method
Constraint hierarchy analysis — explicit exclusions evaluated before surrounding response quality. Cascade attribution: downstream failures traced to the first violated constraint. 6 failure archetypes documented.
Outputs failing explicit exclusion constraints are rejected regardless of response quality elsewhere. Constraint hierarchy is not negotiable.
Prevented constraint-violating outputs from entering downstream preference datasets and corrupting RLHF training signal.
CASE-002 · 2024
RLHF · Hallucination
Plausible-Sounding Fabrication Detection
Hallucination DetectionFact VerificationPattern Analysis
Documented30+ failure patterns
Problem
Structurally convincing responses with embedded subtly fabricated citations and distorted statistics. Correct format, confident tone — surface raters scored them highly.
Method
Entailed vs compatible test: claims entailed by the task are valid; claims merely compatible with it are hallucination flags. 4-type taxonomy built. One confirmed hallucination overrides everything else.
Confident tone is not evidence. Any precise claim not directly supported by the prompt triggers explicit reality check — one confirmed hallucination is an automatic reject.
Prevented hallucinated outputs from entering training datasets, protecting downstream model behaviour from corrupted preference signal.
CASE-005 · 2025
Multimodal · Image-Text
Image-Text Alignment & Visual Grounding Failures
Multimodal EvaluationVisual GroundingHallucination Detection
ProtocolDual-layer evaluation
Problem
Models generating fluent, contextually coherent image descriptions that misrepresented specific visual details, relationships, and spatial arrangements. Invisible to text-only reviewers.
Method
Dual-layer protocol: text coherence evaluated independently from visual grounding accuracy. Mismatch taxonomy: entity misidentification, spatial errors, attribute hallucination, confidence masking.
Textual fluency in multimodal outputs is an unreliable quality proxy. Each visual claim requires independent grounding verification.
Dual-layer verification adopted across subsequent multimodal cycles, preventing visually inaccurate outputs from passing text-only evaluation.
CASE-003 · 2024
Bias · Cross-Cultural QA
Quantifying the Africa Knowledge Gap
Bias DetectionCross-Cultural QAParallel Prompt Design
Measured gap−3.3 pts quality delta
Problem
Model responses about African institutions were systematically lower quality than equivalent Western responses. The gap was invisible to US-centric rubrics — a meta-bias in the evaluation itself.
Method
Parallel prompt battery matched across Western and African contexts in information density and complexity. Identical rubrics applied. Quality tracked by region and knowledge domain.
A 3.3-point gap on matched prompts is not noise — it is a systematic training data failure. Flagged as a data collection priority, not an output quality issue.
Structured evidence directly informed data collection strategy for underrepresented knowledge domains, addressing the training gap at the source.
CASE-004 · 2023
Data Ops · Validation
50K-Point Schema Normalisation
99%+ AccuracySchema AuditPipeline QC
Scale50,000+ entries validated
Problem
Silent schema drift across batches from multiple annotation sources. No hard errors — inconsistent labelling conventions that would corrupt model signal downstream.
Method
Cross-batch consistency audit mapping every deviation category. Normalisation guide aligning all batches to a single standard with documented correction rules per deviation type.
Schema drift with no surface errors is still critical failure — it is silent corruption of training signal. Pass threshold requires cross-batch consistency, not just per-entry accuracy.
Silent schema corruption removed before it could degrade downstream model training signal. Audit framework reused for subsequent annotation batches.
Why This Matters

Evaluation failures
have consequences

⚠
Incorrect evaluations reinforce hallucinations
When bad outputs are scored as good, the model learns the wrong behaviour. Every missed failure is a training signal pointing in the wrong direction.
⚠
Biased rubrics corrupt preference data
Evaluation that can't see its own assumptions doesn't catch the failures it was designed to find. Rubric design is risk surface, not paperwork.
⚠
Silent drift degrades model performance
Schema inconsistency, annotation drift, and missed edge cases accumulate silently. By the time they're visible, the damage is already in the weights.
