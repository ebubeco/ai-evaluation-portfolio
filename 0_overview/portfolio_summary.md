
# Portfolio Summary

I am an AI evaluation and annotation specialist focused on assessing
model outputs for instruction adherence, reasoning quality, and
factual correctness.

My work centers on tasks where responses are often partially correct
but contain subtle errors, requiring careful evaluation and precise
judgment.

## Core Strengths

- Interpreting and enforcing task instructions with high accuracy
- Comparing multiple responses and selecting the best based on
  defined criteria
- Identifying hidden issues such as incomplete answers, logical gaps,
  and hallucinations
- Providing clear, consistent justifications for every evaluation
  decision

## Evaluation Scope

This portfolio includes structured evaluation runs that reflect real
annotation workflows, including:

- Instruction-following evaluation
- Reasoning comparison and ranking
- Hallucination detection
- Ambiguity resolution

Each evaluation is designed to demonstrate consistent decision-making
under conditions where outputs are not obviously correct or incorrect.

## Approach

I apply a systematic evaluation process that prioritizes:

- Strict adherence to instructions
- Completeness of response
- Factual and logical correctness

The goal is to produce reliable, repeatable judgments aligned with
real-world RLHF and model evaluation standards.
